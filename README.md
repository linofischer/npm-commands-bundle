# NPM-commands-bundle

A bunch of simple npms commands.

## Installation

Use npm  to install the package globally.

```bash
npm i -g @linofischer/npm-commands-bundle
```

## Usage

#### Delete node_modules, delete package-lock.json and run npm install:
```
ncb-reinstall-modules
```
